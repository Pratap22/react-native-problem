# react-native-problem

## Setup steps

1. Clone the repo
2. cd folderName
3. Install dependencies

```bash
yarn install
```

4. Run

Android:

```bash
npx react-native run-android
```

iOS

```bash
cd ios
pod install

//After completion

cd ../
npx react-native run-ios
```
